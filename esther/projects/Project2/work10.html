The Plastic Arts

Perhaps better imagined as the GHOST that comes in advance of a particular lettershape, a MetaFont character is defined only by a set of equations rather than hard-coded coordinates and outline shapes. . . . So it is then possible to treat parameters such as aspect ratio, slant, stroke width, serif size, (curlyness!?) and so on as abstracted input values that can change in each glyph definition, creating not a set of set letters, but instead a set of set parameters, any of which can be changed each time the font is rendered. By changing the value at one location in the MetaFont file, a consistent change is produced throughout the entire font.
